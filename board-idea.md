An ESP32 based "badge" board that can be programmed to be used as a wifi/bluetooth enabled audio recorderw
- 55mm x 95mm board
- USB C interface for programming and battery charging, centered on the short edge of the board, extends  3-5mm outside the edge of the board to fit the case
- Integrated SDCARD reader, connected to appropriate pins
- Connector for 3.7v 2000mAH battery
- Fuel gauge chip (potentially MAX17048 ) connected to battery and ESP32 for battery monitoring
- 1x populated I2S MEMS microphone (L channel)
- 1x unpopulated area for second I2S MEMS microphone (R channel)
- 1x populated momentary switch
- 1x unpopulated area for second momentary switch
- All connectors should be on the front of the board
- 2x holes on top outer edge of board so it can be connected to a lanyard and used as a badge without a case
- 1x hole near the battery connector to allow battery to be mounted on the front or back of the board
- Silkcreen board name somewhere it will fit nicely "Prawnd - Private AI Recording"

See github.com/hashtagcyber/prawnd for prototype information

# Prawnd
ESP32 Clone of Plaud

The device has a configuration page where you can enter a URL and API key. When a recording completes, it's automatically uploaded to a webserver
The same page is how you configure the wifi access.

A start/stop button is used to start/stop recording

## Hardware
- 1x Seeed Studio XIAO ESP32C6
- 2x INMP441
- 1x WWZMDiB 6 Pcs Micro SD TF Card Adapter Mini Reader Module 3.3V 6 Pin SPI Interface
- 1x button
